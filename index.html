About the Game

Number Hunter is an engaging number matching game that generates random numbers and finds matches between different groups. Watch as numbers are generated in real-time, with matches highlighted in red and duplicates in gray. Fill all six boxes with unique matching numbers to win!

Choose between two versions: the standard 2-groups game for a balanced experience, or challenge yourself with the 4-groups version for an extreme test of patience and luck.

2 Groups Version

The standard version with two random number groups. Perfect for casual gameplay with a reasonable chance of finding matches (1/2,401 probability).

  • Balanced gameplay
  • Average completion time: ~30 minutes
  • Ideal for regular sessions
Play 2 Groups Version
4 Groups Version

The challenging version with four random number groups. An extreme test with very rare matches (1/5,764,801 probability).

  • Extremely challenging
  • Extended completion time
  • For experimental purposes
